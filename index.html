Post Mortem Review

A Reflection On Our Process Of Making Platformer

Building Breakout was definitely an educational process. It was interesting to learn about how to use SDL, how to work with different types of media, and how to piece everything together as a team into one working project.

Setting up the project seemed to be the most challenging yet important aspect of the project. Making sure to keep track of our textures, surfaces, media files, objects, etc. in our Resource Manager was a difficult process; however, it was good to learn how to properly manage our project in a way that would make our game perform in an optimize and efficient manner. Also, setting up the file structure, in terms of where certain functions and data would be located was another challenge. Perhaps more planning in the beginning would've made this process easier.

If we had more time, we think it would be good to improve the visuals of our games (images, fonts, etc.) as well as adding additional features (e.g. high score, multipliers, different backgrounds / themes). Also adding Resolution or FPS options would be an interesting add that we would put if we had more time.
